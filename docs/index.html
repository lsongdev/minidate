minidate

Minimal Date Library for JavaScript and Node.js

Installation

$ npm install minidate --save

Example

const minidate = require('minidate');

console.log(
  minidate('2020-02-28')
  .addDays(1)
  .toString('yyyy-MM-dd')
);
